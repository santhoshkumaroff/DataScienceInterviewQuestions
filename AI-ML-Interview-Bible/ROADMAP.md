# 🗺️ Complete AI/ML Learning Roadmap for Freshers

> "The journey of a thousand miles begins with a single step." — Lao Tzu

This roadmap is designed to take you from **absolute beginner** to **interview-ready** in AI/ML. Each stage builds on the previous one. Do not skip stages.

---

## Stage 1: Foundation (Weeks 1-4)

### 1.1 Python (Week 1-2)
**What to learn:**
- Variables, data types, loops, conditionals
- Functions, classes, OOP basics
- Lists, tuples, dictionaries, sets
- File I/O, exception handling
- List comprehensions, lambda functions
- Libraries: NumPy (arrays, operations), Pandas (DataFrames, CSV, cleaning)
- Basic data visualization: Matplotlib, Seaborn

**Resources:** W3Schools, freeCodeCamp Python course, Python for Everybody (Coursera)

**Good when:** You can write a Python script to read a CSV, clean null values, and plot a chart without looking up syntax.

### 1.2 SQL (Week 3)
**What to learn:**
- SELECT, WHERE, ORDER BY, GROUP BY
- JOINs (INNER, LEFT, RIGHT, FULL)
- Aggregation: COUNT, SUM, AVG, MAX, MIN
- Subqueries, CTEs
- Window functions: ROW_NUMBER, RANK, DENSE_RANK

**Resources:** Mode Analytics SQL Tutorial, LeetCode SQL problems

**Good when:** You can write a query to find the top 3 highest-paid employees per department.

### 1.3 Statistics & Probability (Week 4)
**What to learn:**
- Mean, median, mode, variance, standard deviation
- Normal distribution, skewness, kurtosis
- Correlation vs causation
- Probability basics: conditional probability, Bayes' theorem
- Central Limit Theorem, p-values, confidence intervals

**Resources:** StatQuest YouTube channel, Khan Academy

**Good when:** You can explain p-value to a non-technical person.

---

## Stage 2: Core Machine Learning (Weeks 5-8)

**What to learn:**
- Types of ML: Supervised, Unsupervised, Reinforcement Learning
- Train/Test split, cross-validation
- Bias-Variance tradeoff
- Overfitting, underfitting, regularization (L1, L2)
- Evaluation metrics: Accuracy, Precision, Recall, F1, ROC-AUC, RMSE, MAE
- Algorithms:
  - Linear Regression, Logistic Regression
  - Decision Trees, Random Forest
  - SVM, k-NN
  - Naive Bayes
  - K-Means Clustering, Hierarchical Clustering
  - PCA (Dimensionality Reduction)
- Feature engineering, feature scaling

**Practice:** Implement algorithms using scikit-learn. Do at least 3-4 projects.

**Resources:** Andrew Ng's ML Course (Coursera), StatQuest YouTube

**Good when:** You can explain how Random Forest works in 2 minutes.

---

## Stage 3: Advanced Machine Learning (Weeks 9-10)

**What to learn:**
- Ensemble methods: Bagging, Boosting (AdaBoost, Gradient Boosting, XGBoost, LightGBM)
- Hyperparameter tuning: GridSearchCV, RandomizedSearchCV
- Handling imbalanced datasets: SMOTE, class weights
- Feature selection techniques
- Time series forecasting basics (ARIMA, Prophet)

**Resources:** Kaggle Learn, XGBoost documentation

**Good when:** You can improve a model's F1 score from 0.7 to 0.85 using proper techniques.

---

## Stage 4: Generative AI (Weeks 11-13)

**What to learn:**
- Neural Networks basics: Perceptron, activation functions, backpropagation
- Deep Learning: CNNs (images), RNNs/LSTMs (sequences)
- Transformers: Attention mechanism, BERT, GPT
- LLMs: Prompt engineering, fine-tuning, RAG (Retrieval Augmented Generation)
- LangChain basics, Hugging Face ecosystem
- Diffusion models (conceptual understanding)
- Evaluation of LLMs: Perplexity, BLEU, ROUGE

**Practice:** Build a chatbot using LangChain, fine-tune a small LLM on custom data.

**Resources:** Hugging Face course, DeepLearning.AI short courses, Andrej Karpathy's "Let's build GPT"

**Good when:** You can build a simple RAG pipeline that answers questions from a PDF.

---

## Stage 5: Big Data & MLOps (Week 14)

**What to learn:**
- Big Data concepts: Why traditional ML fails at scale
- Spark basics (PySpark), Hadoop conceptually
- MLOps basics: Model deployment (Flask/FastAPI), Docker, CI/CD
- Model monitoring, drift detection
- Cloud basics: AWS Sagemaker / GCP Vertex AI (conceptual)

**Resources:** Databricks tutorials, FastAPI docs

**Good when:** You can deploy a simple model as an API endpoint.

---

## Stage 6: Data Structures & Algorithms (Week 15)

**What to learn:**
- Arrays, strings, hash maps
- Two pointers, sliding window
- Recursion, backtracking
- Trees (BST, binary tree traversals)
- Graphs (BFS, DFS)
- Sorting & searching algorithms
- Dynamic Programming basics (Fibonacci, knapsack)

**Practice:** Solve 50-60 LeetCode problems (Easy + Medium). Focus on Python.

**Resources:** NeetCode YouTube, LeetCode, Grokking the Coding Interview

**Good when:** You can solve a Medium LeetCode problem in 20-30 minutes.

---

## Stage 7: Projects (Weeks 16-17)

**Build 3 solid projects:**

1. **Beginner:** House Price Prediction (regression) — Linear Regression, Random Forest
2. **Intermediate:** Customer Churn Predictor (classification) — XGBoost, handling imbalance
3. **Advanced:** Chat with PDF using RAG (GenAI) — LangChain, FAISS, Streamlit

**Host everything on GitHub** with good README files. Deploy at least one using Streamlit or FastAPI.

---
